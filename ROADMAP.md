# Roadmap

Short-term ideas:

- `prompt-search show <doc_id>` to print full content and metadata for a single match.
- `prompt-search open <doc_id>` to open the source JSONL file at the matching line.
- Search filters: `--since`, `--until`, `--cwd`, `--session-id`.
- Optional incremental FTS maintenance if DuckDB supports it in the future.
